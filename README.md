## Descripción
Aplicación de consola desarrollada en *Python 3.x* que gestiona información sobre países.  
Permite cargar, actualizar, buscar, filtrar, ordenar y calcular estadísticas a partir de un archivo CSV.

## Funcionalidades
- Lectura y escritura de archivos CSV.  
- Agregar y actualizar países.  
- Búsqueda parcial por nombre.  
- Filtros por continente, rango de población o superficie.  
- Ordenamientos por nombre, población o superficie (ascendente/descendente).  
- Estadísticas:
  - País con mayor y menor población.  
  - Promedio de población y superficie.  
  - Cantidad de países por continente.

## Requisitos
- Python 3.8 o superior.  
- No requiere librerías externas.
